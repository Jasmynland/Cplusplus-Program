# Airgead Banking App – Investment Calculator

## Project Summary
This project is a C++ investment calculator that allows users to input an initial investment amount, monthly deposit, annual interest rate, and number of years. The program calculates and displays the investment growth both with and without monthly deposits. The goal of the project was to demonstrate object-oriented programming principles, input validation, formatted output, and financial calculation logic.

## What I Did Well
I believe I did well organizing my code using classes and separating responsibilities between the header and implementation files. I focused on making the program readable and structured, and I made sure the output was formatted clearly for the user. I also ensured the calculations were accurate and matched the project requirements.

## Areas for Improvement
If I were to enhance this project further, I would improve input validation to make the program more secure and user-friendly. I would also refactor parts of the calculation logic to reduce repetition and possibly add error handling for invalid data types. These changes would make the program more efficient and robust.

## Challenges and How I Overcame Them
The most challenging part of this project was structuring the class correctly and ensuring that the interest calculations updated properly each year. I overcame this by reviewing class structure examples, breaking the problem into smaller steps, and testing calculations manually to confirm accuracy.

## Transferable Skills
This project strengthened my understanding of object-oriented programming, class design, modular code organization, and debugging in C++. These skills are transferable to future software development projects and backend programming roles.

## Maintainability and Readability
I structured the program using separate files for clarity and used descriptive variable names. Keeping the calculations inside the class improves maintainability and allows for future expansion without major restructuring.
